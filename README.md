# Radar Touch
#### 概要
tmlib.jsで作った海戦ゲーム風の位置当てゲームです。

#### 説明
* 10×10のマス目のいずれかをタッチして、隠れた宇宙船の位置を当てます。
* 宇宙船は、2マス分、3マス分、そして4マス分の3種類があり、それぞれのマスの数分を当てなければ破壊できません。
* 画面上部に表示されている爆弾の数以内で、3種類全ての宇宙船を破壊できれば勝利です。
* ゲームは5回戦です。
